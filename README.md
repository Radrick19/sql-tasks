# 📌 Сессионное задание — Итоговый проект: База данных

## 📖 Описание проекта

В рамках данного проекта реализованы SQL-запросы для анализа данных в нескольких предметных областях:

- Транспорт (автомобили, мотоциклы, велосипеды)  
- Автомобильные гонки  
- Гостиницы и бронирования  
- Организационная структура компании (сотрудники, проекты, задачи)  

Цель проекта — продемонстрировать навыки работы с SQL, включая:

- сложные `JOIN`
- агрегации (`GROUP BY`, `HAVING`)
- оконные функции и их альтернативы
- рекурсивные запросы (`WITH RECURSIVE`)
- работу с иерархиями данных
- строковые агрегаты (`STRING_AGG`)
